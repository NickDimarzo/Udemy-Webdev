HTML Tables

General Notes

  • 2 dimensional
  • used for data
  • created with rows and columns
  • used to compare different data

Tables: TR, TD, TH elements

Notes:

  • TD = Table Data
    • represents a single cell
  • TR = Table Row
    • groups table data (TD) into a single row in the table
  • TH = Table Header
    • Defines a heading in the table

Tables: Thead, Tbody, Tfoot elements

Notes:

  • Thead
    • Defines the set of rows that are the head columns of the table
  • Tbody
    • Contains the set of tables rows that comprise the actual data of the table
  • Tfoot
    • Not always needed
    • Can be used for tables Totals

Tables: Colspan and Rowspan

Notes:

  • Colspan
    • Allows data to take up more than one cell in the column
  • Rowspan
    • Allows data to take up more than one cell in the row
